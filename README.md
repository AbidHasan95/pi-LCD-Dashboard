
# LCD Dashboard

A simple lcd screen to keep track of Time, Date, Temperature (fetched using a weather API) 



## Things Used
- Raspberry Pi 3 model B
- LCD 16x2 with I2C module
- 10k Potentiometer
- Jumper cables
