
# LCD Dashboard

A simple lcd dashboard to keep track of Time, Date, Temperature (fetched from a weather API) etc.



## Things Used
- Raspberry Pi 3 model B
- LCD 16x2 with I2C module
- 10k Potentiometer (For adjusting the backlight)
- Jumper cables
